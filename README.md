# rust-image-by-json
Written in Rust. Reads JSON file to batch update images

## Cargo Instructions

Example: Load basic.json
````
 cargo run -- --json-file-path ./test-json/basic.json
````

Example: Load basic_batch.json
````
 cargo run -- --json-file-path ./test-json/basic_batch.json
````

Example: Load basic_grayscale.json
````
 cargo run -- --json-file-path ./test-json/basic_grayscale.json
````
